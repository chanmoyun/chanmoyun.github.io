Win10 企业版 LTSC

写在前面

你好，我是禅墨！

一直嫌弃Win10系统繁杂，好多我用不到的东西，系统本身占用内存二十多G，然后系统运行感觉没那么流畅，时不时出现bug，最让我难受的就是系统一直更新，时不时地给你来个蓝屏，那叫一个痛苦啊。然后我偶然在网上看到了Win10 LTSC 版，然后就不停地去搜，详细了解之后，当即决定：换！（我那么多软件，今天才彻底装完）用过这两天，只想说：真香！所以决定不再私藏，分享给大家
今日推荐歌曲：

系统预览



聊聊优点

好用，那是相当顺溜，顺到飞起，这么说吧，之前开机启动需要7-8秒，现在开机4-5秒，动画响应超快，系统内存占用方面装完只有9G(安全引导下删除了旧版残留)，然后进行更新，安装必要软件（git node.js edge chrome vs code 某流氓360）后也就占有21G，桌面给挪动到了 D盘。功能方面 Win10 专业版功能一个没少，小娜，微软应用商店，Xbox的基本用不到的都没了。由于是给企业提供的，所以在安全上更为可靠，没有频繁的更新（半年一次），但是日常的系统补丁仍然会有，系统极限运行内存是256M(虚拟机实测)，128M就会影响一些功能，所以对一些老机器十分友好。

说说缺点

当然有优点也会有缺点，由于过于精简，自带浏览器只有IE(关于这个浏览器，我也不说啥，都懂)，需要自行安装，如果是普通用，建议安装最新的Microsoft Edge，编程开发的根据需要再装个Chrome(比如我)
还有就是激活方面，每次激活只能用180天，时间到再激活就行了，也就两分钟的事情

再谈谈Microsoft Edge

最新发布的ME 可不在是以前的ME，本次抛弃了IE内核，完全继承了chrome内核，然后做了许多优化，不再是之前的转圈响应，目前使用来看，比Chrome更丝滑(毕竟是亲儿子)，支持插件的安装，页面整洁，没有广告植入(搜索引擎选百度就另说了)
